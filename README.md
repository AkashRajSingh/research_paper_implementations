# Research Paper Implementations

A collection of **PyTorch implementations** of influential research papers, built to bridge the gap between theory and code.

Each implementation includes:

*  PyTorch code in notebooks
*  A reference PDF containing my notes, key explanations, and important snippets from the original paper
